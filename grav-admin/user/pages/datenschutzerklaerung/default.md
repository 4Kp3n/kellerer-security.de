##### 2. Server-Logs
Beim Aufruf der Website erfasst der Server die Verbindungsdaten der Anfragen. Diese Informationen werden temporär in sogenannten Logfiles gespeichert.

Folgende Informationen werden dabei erhoben:
* IP-Adresse des anfragenden Rechners  
* Datum und Uhrzeit des Zugriffs  
* Name und URL der abgerufenen Datei  
* Website, von der aus der Zugriff erfolgt (Referrer-URL)  
* verwendeter Browser und ggf. das Betriebssystem des Rechners sowie der Name des Access-Providers  

Im Moment findet keine Auswertung dieser Daten statt.
